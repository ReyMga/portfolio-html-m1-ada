Curso de desarrollo Frontend

Un programador Frontend debe saber de códigos [HTML, CSS y JavaScript](HTML, CSS y JavaScript) para poder usar algunos frameworks o librerías que expanden sus capacidades para crear cualquier tipo de interfaces de usuarios. React, Redux, Angular, Bootstrap, LESS, Sass, Stylus y PostCSS son algunos de ellos.

  2. Introducción a CSS

  3. Introducción a Sass

  4. Introducción a JavaScript
